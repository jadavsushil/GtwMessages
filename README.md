# Messages plugin for CakePHP

Plugin for Internal Messages with cakephp.

## Requirements

CakePHP 2.4.0+   
[GtwUsers](https://github.com/Phillaf/GtwUsers) and its dependencies properly configured

## Features
* Compose Message
* Inbox Message
* Sent Message
* Trash Message
* Reply Message
* Forward Message


## Installation

	Copy this plugin in a folder named `app/Plugin/GtwMessages`

Load the plugin

	CakePlugin::load('GtwMessages');
